# 📚 The Bookery

**A low-bandwidth digital library providing free access to books for South Africans.**

Built with accessibility, simplicity, and affordability in mind, The Bookery is designed for users with limited internet access while still delivering a clean and modern browsing experience.

## 🧭 Project Overview

The Bookery is a digital library platform aimed at improving access to literature across South Africa, particularly for communities with limited internet connectivity or restricted access to physical libraries.

The website prioritizes:

- Low bandwidth usage
- Minimal page weight
- Lightweight downloads (ePub format)
- Clear navigation and strong contrast

The visual design takes inspiration from Cape Town, using a calm, minimal colour palette that reflects the city’s coastal identity.

---

## 🎯 Goals & Objectives

- **Improve Access**  
  Provide free digital books in English and isiZulu through a low-bandwidth platform.

- **Community Engagement**  
  Host monthly virtual Q&A sessions with South African authors.

- **Partnership Growth**  
  Partner with at least 10 schools in Khayelitsha within one year.

The broader goal is to improve literacy and preserve access to influential literature for current and future generations, regardless of socioeconomic or technological limitations.

---

## ✨ Features

- Site-wide animated logo
- Consistent navigation across all pages
- Google Calendar integration for events
- Library organised into shelves with visible book covers
- Lightweight ePub downloads
- High-contrast text and buttons for accessibility
- Mobile-responsive design
